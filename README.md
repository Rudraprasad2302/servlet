# 🌐 Java Servlet Project

A basic web application built using **Java Servlets** to understand server-side programming and dynamic web content generation.

This project was developed during my initial learning phase of Java Web Development.

---

## 📌 Project Overview

The Servlet project demonstrates how HTTP requests and responses are handled in Java using the Servlet API.

It helped me understand:

- Client-Server architecture
- HTTP request & response lifecycle
- Form data handling
- Web application deployment
- Servlet configuration

---

## ✨ Features

- 📥 Handles HTTP GET and POST requests
- 📨 Processes form data
- 📄 Generates dynamic responses
- 🌐 Basic web application structure
- ⚙️ Web.xml configuration (if used)

---

## 🛠 Tech Stack

- **Language:** Java
- **Technology:** Java Servlet API
- **Server:** Apache Tomcat
- **Frontend:** HTML (basic forms)
- **Architecture:** Client-Server Model

---


## ⚙️ How It Works

1️⃣ User submits data through an HTML form.  
2️⃣ Servlet receives the request.  
3️⃣ Data is processed on the server.  
4️⃣ Response is generated dynamically and sent back to the browser.

---

## ▶️ How to Run the Project

### 1️⃣ Install Apache Tomcat

Download and configure **Apache Tomcat**.

### 2️⃣ Import Project into IDE

- Open Eclipse / IntelliJ
- Import as Dynamic Web Project

### 3️⃣ Configure Server

- Add Tomcat Server
- Deploy project to server

### 4️⃣ Run Application

Start the server and open:

```
http://localhost:8080/ProjectName/
```
